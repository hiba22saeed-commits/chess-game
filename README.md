##Chess Game (C++)

This project is a console-based implementation of the classic chess game, developed in C++ using object-oriented programming principles. It recreates a complete two-player chess experience on an 8×8 board, where each piece follows its actual movement rules. The system ensures that only valid moves are allowed and automatically prevents illegal actions during gameplay.

The program manages turn-based interaction between players and continuously evaluates the game state to detect conditions such as check, checkmate, and stalemate. Its modular structure, with separate classes for the board, pieces, and overall game control, makes the code organized, readable, and easy to maintain. This project highlights the practical application of core OOP concepts like inheritance, encapsulation, and polymorphism while implementing real-world game logic.

##Features
Console-based 8×8 chessboard display
Accurate movement rules for all chess pieces
Turn-based two-player gameplay
Move validation and illegal move handling
Detection of check, checkmate, and stalemate
Technologies Used
C++
Object-Oriented Programming (OOP)
