# Chess Game (C++)

This project is a console-based implementation of the classic chess game, developed in C++ by applying all major pillars of Object-Oriented Programming (OOP). The system provides a complete two-player chess experience on an 8×8 board, where each chess piece follows its standard movement behavior and gameplay rules.

The project is designed using a modular and object-oriented architecture that improves code organization, scalability, and maintainability. Core OOP concepts including **Encapsulation, Inheritance, Polymorphism, and Abstraction** are extensively utilized throughout the system.

- **Encapsulation** is used to secure and manage game data within classes.  
- **Inheritance** allows different chess pieces to share common behaviors from a base class.  
- **Polymorphism** enables dynamic handling of piece movements and actions.  
- **Abstraction** simplifies complex game logic through well-structured class design.  

The game continuously validates player moves and prevents illegal actions during gameplay. It also monitors the board state to detect important conditions such as **Check, Checkmate, and Stalemate**, ensuring an authentic chess-playing experience.

This project demonstrates the practical implementation of OOP concepts in solving real-world problems and building interactive console-based applications.

---

## Features

- Console-based 8×8 chessboard interface  
- Accurate movement rules for all chess pieces  
- Turn-based two-player gameplay  
- Move validation and illegal move prevention  
- Detection of Check, Checkmate, and Stalemate  
- Modular class-based architecture  
- Efficient game state management using OOP principles
